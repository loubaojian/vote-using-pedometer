## Vote Using Cordova Plugin Pedometer

## Install

At www/
```
$ bower install
```

```
$ cd ..
$ cordova plugin add cordova-plugin-pedometer/
```

```
$ cordova platform add ios
$ cordova platform add android
```

## Run

For iOS, run platforms/ios/VoteApp.xcodeproj

For android,
```
$ cordova run android --device
```
